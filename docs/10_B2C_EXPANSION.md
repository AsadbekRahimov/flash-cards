# 📋 LexiFlow Pro — Анализ текущего функционала и план расширения до B2C SaaS

> Документ составлен на основе изучения `docs/00_README.md` … `docs/09_AI_AGENTS_GUIDE.md` и `PROGRESS.md` (на 2026-04-24).
> Цель — превратить B2B-платформу для учебных центров в массовый продукт с подпиской для всех желающих.

## 1. Текущий функционал LexiFlow Pro

### 1.1 Позиционирование сейчас
Платформа **B2B для языковых центров**. Студенты попадают в систему только через Telegram-группу центра, учитель запускает тренировки и экзамены командами бота. Монетизации нет — MVP без платежей.

### 1.2 Что уже реализовано (Sprints 1–9, ~98% MVP)

**Telegram Bot**
- Group Lock (белый список `chat_id`), webhook с 2-секретами, rate-limit.
- Команды учителя: `/start_training`, `/start_exam`, `/close_exam`, `/stats`, `/help`.
- События `my_chat_member` / `new_chat_members` → авто-регистрация студентов.

**Telegram Web App (TWA, Vue 3 + Pinia)**
- Авторизация через Telegram `initData` (HMAC-SHA256) + JWT 15 мин.
- **Тренировка (Flashcards):** карточка → перевод/пример → 3 оценки (😰🙂😎), TTS (Web Speech API + edge-tts fallback).
- **Экзамен:** MCQ из 1 слова + 4 вариантов, таймер сессии и вопроса, подсчёт баллов (×1.5 за hard-слова, штраф за время), лидерборд в чат группы.

**SRS (SM-2)**
- Таблица `word_repetitions`: `repetitions`, `interval_days`, `easiness_factor`, `next_review_at`, `is_hard`.
- Cron 18:00 — пуш в ЛС: «Пора повторить N слов».

**Контент**
- JSON-импорт через Filament (schema `stage → lesson → vocabulary[]`), транзакционный upsert, лимиты 2 MB / 200 слов.
- CRUD по дереву Stage → Lesson → Word (без удаления — сохранение истории).

**Админка (Filament 5.6)**
- CRUD users/groups/lessons/words, роли spatie/permission.
- Дашборд: студенты, экзамены, accuracy, топ-10 активных, топ-20 трудных слов.

**Безопасность (Sprint 9)**
- HTTP security headers, IP allowlist webhook, маскирование PII в логах, XSS-защита, тесты на race-conditions. 117 тестов Pest проходят.

### 1.3 Что НЕ сделано (Sprint 10 + out-of-scope)
- HTTPS Let's Encrypt, 2FA админов, Sentry, бэкапы, Lighthouse, coverage.
- **Нет монетизации, нет подписок, нет самостоятельной регистрации, нет личного кабинета без группы.**

### 1.4 Что блокирует B2C прямо сейчас

| Блок | Как работает сейчас | Проблема для B2C |
|---|---|---|
| Регистрация | только приглашение в Telegram-группу центра | пользователь не может сам завести аккаунт |
| Модель данных | `students.telegram_group_id` обязателен, один юзер в двух группах = 2 профиля | нет кросс-группового профиля |
| Контент | заливает только admin центра | нет self-serve уроков и своих колод |
| Учитель | обязательное звено для запуска тренировки/экзамена | без учителя ничего не стартует |
| Лидерборд | только внутри группы | нет глобального соревнования |
| Платежи | отсутствуют | нет подписок, триалов, биллинга |

## 2. Что нужно добавить для B2C SaaS с подпиской

### 2.1 Фундамент — новая модель пользователя (обязательно)

- **Роль `subscriber`** в дополнение к admin/teacher/student.
- `users` ← добавить `auth_type` (telegram/email), `subscription_status`, `trial_ends_at`.
- **Self-serve онбординг через Telegram `/start` в ЛС**: бот сам создаёт `personal_workspace` (как «группа из одного»), даёт trial 7 дней.
- Отвязать `students` от обязательного `telegram_group_id` → ввести `learner_profiles` с единым прогрессом, поверх которого можно привязывать многие группы.
- Миграция: **`telegram_group_id` → nullable**, создать `virtual_group` типа `personal` для соло-подписчиков.

### 2.2 Биллинг и подписки

- **Laravel Cashier + Stripe** (или ЮKassa / Tinkoff / Payme / Click для СНГ-рынка).
- Тарифы:
  - **Free** — 50 слов/день, 1 урок, без экзаменов.
  - **Pro ($5–9/мес)** — безлимит слов, все уроки, голос, экспорт, streak-фризы.
  - **Pro+AI ($14–19/мес)** — AI-генерация уроков из своих текстов, персональный план, озвучка Azure / ElevenLabs.
  - **Family (3 slot)** и **Teacher** (для репетиторов-одиночек, до 20 учеников).
- Оплата внутри Telegram: **Telegram Payments API** (Stripe / ЮKassa provider) — платёж прямо в боте, без редиректа.
- Webhooks → `invoice.paid` → `subscription.active` + `grace_period` для неудачных списаний.

### 2.3 Контент — чтобы платили не за урок центра, а за библиотеку

- **Публичный каталог курсов** в TWA: Beginner A1 → Proficiency C2, IELTS, TOEFL, Business, Travel, Tech, Medical.
- **Пользовательские колоды**: «Мои слова», импорт из CSV / Anki `.apkg`, добавление слова из браузерного extension (Chrome / Edge).
- **Импорт из текста**: вставил статью / YouTube-субтитры → AI выделил N новых слов → создал колоду (Claude API / GPT-4o-mini).
- **Community decks**: пользователи могут шарить колоды (модерация), рейтинги, downloads — мощный organic-growth канал.

### 2.4 AI-фичи (главный джастификатор цены Pro+)

- **AI Tutor в чате**: диалог с ИИ на английском по теме урока, проверка грамматики, разбор ошибок. Claude Haiku 4.5 дёшево и быстро.
- **Генерация упражнений**: fill-in-the-blank, match, listening (TTS), writing — по словарю урока автоматически.
- **Speaking practice**: пользователь записывает аудио → Whisper транскрипция → сравнение с эталоном, подсветка ошибок произношения.
- **Персональный план**: AI анализирует прогресс, streak, трудные слова → выдаёт «сегодня: 10 новых + 20 повторов + 1 диалог».
- **Контекстные примеры**: для каждого слова — 3 примера в контексте уровня пользователя.

### 2.5 Геймификация и удержание (retention = LTV)

- **Streak** с freeze-днями (как Duolingo) — платная фича.
- **Глобальный и лиговой лидерборды** (Bronze → Diamond лиги, ротация раз в неделю).
- **Достижения**: 100 слов, 30 дней, первая 5-ка на экзамене, и т.д.
- **Daily goal** (5 / 10 / 20 слов) + пуш в Telegram + трекер в TWA.
- **Друзья**: `/invite @username` → взаимные челленджи, сравнение streak.

### 2.6 Расширение форматов обучения (за пределами flashcards)

- **Аудирование**: короткие диалоги (30–60 сек) с transcript, задания по смыслу.
- **Чтение**: адаптивные тексты по уровню CEFR, клик по слову → мгновенный перевод + «добавить в колоду».
- **Грамматика**: мини-уроки + проверочные тесты с объяснениями ошибок.
- **Письмо**: задания «напиши 5 предложений со словом X» → AI-фидбек.
- **Shadowing**: проигрывание фразы → повтор пользователем → оценка произношения.

### 2.7 B2B2C — оставить путь для репетиторов и микрогрупп

Не выкидывай B2B — расширь:

- **Teacher Plan**: репетитор-одиночка платит $19–29/мес, получает до 20 учеников, свою реферальную ссылку → ученик регится сам, прогресс виден учителю.
- **School Plan** (старый B2B): ≥50 учеников, свой брендинг TWA, экспорт отчётов, SSO.
- Это даёт **три источника выручки**: B2C подписчики, репетиторы, центры.

### 2.8 Платформа — дальше только Telegram вас тормозит

- **PWA / отдельный web.lexiflow.app** — тот же Vue 3 bundle, Telegram Login Widget, запуск без Telegram. Снимает ограничение «только Telegram-юзеры».
- **Mobile app (Capacitor / React Native)** на базе существующего TWA — в App Store / Google Play. Очень расширяет TAM.
- **Browser extension**: выделил слово на любой веб-странице → «добавить в колоду LexiFlow» + мгновенный перевод. Сильный acquisition-канал.

### 2.9 Маркетинг и growth (инфра)

- **Рефералка**: пригласил друга → +14 дней Pro обоим.
- **Affiliate**: 30% от первого платежа лид-магнита (репетиторы, блогеры-преподаватели).
- **SEO-лендинги** по темам: «Топ-500 слов IELTS», «Словарь для программистов» → бесплатная колода в обмен на email / Telegram.
- **Публичные профили** `lexiflow.app/u/username` — streak, достижения (вирально).

### 2.10 Аналитика и операционка

- **Product analytics**: PostHog / Mixpanel — воронки онбординга, churn по тарифам.
- **Customer.io / Loops** для email / Telegram-рассылок (reactivation, upsell).
- **Admin dashboard расширить**: MRR, churn, trial→paid conversion, LTV, CAC.
- **Support**: виджет в TWA + Telegram-бот поддержки с AI-FAQ (Claude RAG по docs).

## 3. Примерный roadmap v2 (B2C переход)

**Фаза 1 (4–6 недель) — фундамент:**
1. Миграция БД (nullable `telegram_group_id`, `learner_profiles`, `subscriptions`).
2. Self-serve `/start` онбординг в ЛС + personal workspace.
3. Cashier + Telegram Payments, 2 тарифа (Free, Pro), 7-дней trial.
4. Публичный каталог из 5–10 курсов (перенос существующего контента).

**Фаза 2 (4–6 недель) — контент и удержание:**

5. Пользовательские колоды + CSV / Anki импорт.
6. Streak, daily goal, достижения, глобальный лидерборд.
7. Чтение + аудирование режимы.
8. Реферальная программа.

**Фаза 3 (6–8 недель) — AI и рост:**

9. AI Tutor, генерация упражнений, AI-план (Claude Haiku 4.5).
10. Speaking (Whisper + pronunciation scoring).
11. PWA вне Telegram + browser extension.
12. Tier Pro+AI, affiliate-программа.

**Фаза 4 — B2B усиление:**

13. Teacher Plan + School Plan, брендинг, SSO, отчёты.
14. Mobile-приложения.

## 4. Ключевой вывод

Текущая архитектура **чистая и расширяемая** (Laravel 12 + сервисный слой, SM-2 изолирован, TWA — отдельный фронт), поэтому переход на B2C — это в первую очередь **добавление поверх**, а не переписывание. Основной блокер — жёсткая связка `student ↔ telegram_group`; её надо ослабить в первую очередь, остальное наращивается поэтапно.
